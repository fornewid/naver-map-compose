Parameters

position

좌표를 지정합니다. 만약 position이 유효하지 않은(LatLng.isValid가 false인) 좌표라면 InvalidCoordinateException이 발생합니다.

bearing

방위를 지정합니다. 방위가 북쪽일 경우 0도이며, 시계 방향으로 증가합니다.

icon

아이콘을 지정합니다.

iconWidth

아이콘의 너비를 지정합니다. LocationOverlayDefaults.SizeAuto일 경우 이미지의 너비를 따릅니다.

iconHeight

아이콘의 높이를 지정합니다. LocationOverlayDefaults.SizeAuto일 경우 이미지의 높이를 따릅니다.

anchor

아이콘의 앵커를 지정합니다. 앵커는 아이콘 이미지에서 기준이 되는 지점을 의미합니다. 앵커로 지정된 지점이 오버레이의 좌표에 위치합니다. 값의 범위는 (0, 0)~(1, 1)이며, (0, 0)일 경우 이미지의 왼쪽 위, (1, 1)일 경우 이미지의 오른쪽 아래를 의미합니다.

subIcon

보조 아이콘을 지정합니다. null일 경우 보조 아이콘이 그려지지 않습니다.

subIconWidth

보조 아이콘의 너비를 지정합니다. LocationOverlayDefaults.SizeAuto일 경우 이미지의 너비를 따릅니다.

subIconHeight

보조 아이콘의 높이를 지정합니다. LocationOverlayDefaults.SizeAuto일 경우 이미지의 높이를 따릅니다.

subAnchor

보조 아이콘의 앵커를 지정합니다. 앵커는 보조 아이콘 이미지에서 기준이 되는 지점을 의미합니다. 앵커로 지정된 지점이 오버레이의 좌표에 위치합니다. 값의 범위는 (0, 0)~(1, 1)이며, (0, 0)일 경우 이미지의 왼쪽 위, (1, 1)일 경우 이미지의 오른쪽 아래를 의미합니다.

circleRadius

원의 반경을 지정합니다. 반경이 0일 경우 원이 그려지지 않습니다.

circleColor

원의 색상을 반환합니다.

circleOutlineWidth

원의 테두리 두께를 지정합니다. 0일 경우 테두리가 그려지지 않습니다.

circleOutlineColor

원의 테두리 색상을 지정합니다.

tag

태그를 지정합니다. 기본값은 null입니다.

minZoom

오버레이가 보이는 최소 줌 레벨을 지정합니다. 기본값은 NaverMapConstants.MinZoom입니다.

minZoomInclusive

지도의 줌 레벨과 오버레이의 최소 줌 레벨이 동일할 때 오버레이를 보일지 여부를 지정합니다. 만약 inclusive가 true이면 오버레이가 나타나고 false이면 나타나지 않습니다. 기본값은 true입니다.

maxZoom

오버레이가 보이는 최대 줌 레벨을 지정합니다. 기본값은 NaverMapConstants.MaxZoom입니다.

maxZoomInclusive

지도의 줌 레벨과 오버레이의 최대 줌 레벨이 동일할 때 오버레이를 보일지 여부를 지정합니다. 만약 inclusive가 true이면 오버레이가 나타나고 false이면 나타나지 않습니다. 기본값은 true입니다.

zIndex

보조 Z 인덱스를 지정합니다. 전역 Z 인덱스가 동일한 여러 오버레이가 화면에서 겹쳐지면 보조 Z 인덱스가 큰 오버레이가 작은 오버레이를 덮습니다. 기본값은 0입니다.

globalZIndex

전역 Z 인덱스를 지정합니다. 여러 오버레이가 화면에서 겹쳐지면 전역 Z 인덱스가 큰 오버레이가 작은 오버레이를 덮습니다. 또한 값이 0 이상이면 오버레이가 심벌 위에, 0 미만이면 심벌 아래에 그려집니다.

onClick

클릭 이벤트 리스너를 지정합니다. 사용자가 오버레이를 클릭하면 호출됩니다. 오직 클릭 이벤트 리스너가 지정된 오버레이만이 클릭 이벤트를 받을 수 있습니다. 예를 들어 마커와 지상 오버레이가 겹쳐져 있고 지상 오버레이에만 클릭 이벤트 리스너가 지정된 경우, 사용자가 마커를 클릭하더라도 지상 오버레이가 클릭 이벤트를 받습니다.
