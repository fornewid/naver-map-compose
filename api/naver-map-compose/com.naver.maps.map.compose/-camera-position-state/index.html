CameraPositionState

class CameraPositionState(position: CameraPosition = NaverMap.DEFAULT_CAMERA_POSITION)

지도의 카메라 상태를 제어하고 관찰할 수 있는 상태 개체입니다. CameraPositionState는 단일 MapView의 상태를 반영하므로 한 번에 하나의 NaverMap composable에서만 사용할 수 있습니다.
